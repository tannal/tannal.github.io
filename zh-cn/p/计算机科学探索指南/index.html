编写程序

编译器

编译器，把结构化，带有描述性质的文字（通常是人写的或者AI生成的）翻译成汇编或机器语言（x86，arm，rsicv，mips）的叫编译器，最后没有看到机器代码的叫解释器，大部分编译器都会有中间表示如SSA或者Sea of Nodes。

操作系统

操作系统，创造一堆概念和服务，帮你管理CPU和内存和硬盘和显示器和鼠标和键盘和显卡和麦克风和扬声器和摄像头和网卡和声卡等各种各样的能插在电脑上面的东西，作为用户你需要的时候按照操作系统的说明书让他帮你完成即可，操作系统是电脑的管理者，你只是个用户，而操作系统听root用户的，你是root的话约等于你拥有这台电脑。

计算机的Scalability问题

Scalability问题引发的分布式系统和虚拟化的研究，以及这些研究的商业成果云计算。

分布式系统，一台电脑就算买最好的CPU，也扛不住双十一，所以我们需要好多电脑，叫机房，再多一点叫数据中心，我怎么把应用程序运行在1000台电脑上面还能保证不出错，同时获得尽可能现行的1000倍速度提升？

我今天有100台电脑，如何让我的产出达到100倍的线性关系，有1000台电脑产出到1000倍，这就是scalability的问题，于是出现了虚拟化技术，以及其商业化版本——云计算（我发现我的数据中心有一半的电脑在空闲，多浪费，不如租出去）。

计算机科学历史

二战后 1950

密码破译，编码，纠错码

半导体行业 1970-1980

这期间得到大力发展的是编译器汇编器，以及操作系统，C语言等高级编译器。

隶属于美国通用电气（AT&T）的贝尔实验室开发出了unix操作系统，并广泛应用于高校和研究中心。主要人物是Ken和Dennis，在设计实现unix操作系统的过程中创造了C语言

Xerox研究中心个人电脑时代 1980 - 1990

这段期间得到大力发展的是计算机科学中的编程语言（主要是），操作系统，以及互联网（下一个时代）。

开端是Xerox研究中心的以下成果之一，用户界面被Steve Jobs学习并进一步发展。

  • 用网络连接不同的电脑，即互联网
  • 用户界面
  • 面向对象编程

诞生了Mac电脑和Windows操作系统，在乔布斯的Lost Interview采访中他表示Web将会改变世界，并说苹果会做到这一点。，

采访是在1995年，而Webkit第一版发布是在1998年，很可能在采访的1995年苹果就已经fork了KDE的浏览器引擎开始在做webkit了，在这也落在互联网泡沫的中期。

互联网时代 1990-2010

在个人电脑时代后期，IBM出售了个人电脑业务给联想。

这段期间得到大力发展的是浏览器和万维网，由于Netscape实在无法抵抗微软内置IE抢占市场的商业行为，开源了firefox的源代码，也是开源操作系统内核Linux发展的黄金时期，面向对象编程语言如Java在此期间也开始喷涌而出。

当时软件行业以售卖为生（类似于现在的游戏行业），再加上天下苦微软和AT&T久矣，firefox源代码的开放间接导致了世界范围内的开源软件运动（Free Software MoveMent）。

GNU组织诞生，重写大部分unix系统提供的软件包，同时也想要重写一个开源的操作系统，后因进展缓慢转向由芬兰大学生发起的hobby project —— Linux。

大量的用户自发的开始把能够在windows和unix上运行的软件移植到Linux，最著名的也是最终导致Linux成功的是Apache httpd服务器。人们可以免费的获取Linux，免费的安装httpd来搭建自己的网站服务，这让Linux的用户量随着互联网泡沫一起攀升。

由Netscape发起的浏览器项目得到大量关注，搜索引擎和门户网站等商业公司成长起来，诞生了谷歌，脸书，Netflix，亚马逊等网络服务，Oracle。

同样来自斯坦福的Sun Micosystems创业公司，这是一家技术公司的典范，自己设计CPU，自己开发编译器，自己开发操作系统，自己开发编程语言Java，最后被Oracle收购了，Sun的CTO是前Google CEO 艾瑞克·施密特。

在中国大陆诞生了老牌互联网企业，百度，腾讯，阿里巴巴。

2D图形 GUI是怎么整出来的？

首先我们有平台相关图形API，OpenGL，Vulkan，DirectX，Metal。

然后我们有平台相关的窗口API，用于创建窗口和渲染区域（这部分的内容通常由图形API确定），这个程序帮我们handle各种各样的事件，如鼠标移动（会伴随着碰撞测试）。

我们用图形API，写一些shader和渲染逻辑，得到一个2D图形库，这个2D图形库的API跟浏览器的Canvas类似。

  • 三角形
  • 长方形
  • 圆形
  • 渲染一个路径，多边形
  • 渲染字体
  • 渲染image

市面上的2D图形库有

  • Skia 被谷歌收购，用于chrome浏览器和
  • Cairo Webkit使用，不过可能要部分切成skia。
  • Webrender servo项目和Rust项目诞生的专注于web渲染，应该来自于mozilla图形团队。
  • Vello 由Google Fonts投资，Android。
  • 没了

2D图形库加上对于窗口API的封装，就是一个GUI toolkit，如

  • GTK
  • QT
  • Flutter 使用skia。
  • Electron 这玩意就是个chrome，使用skia，对你没听错，VSCode是个浏览器。
  • Java 生态，用Java写GUI最著名的是？对没错，Jetbrains的IDE。
  • Android应用编程，Kotlin以及KMP
  • React Native

有了GUI toolkit加上一些库和包和组件，基本上就能拿来做生产级应用了。

比较重要的库是

  • 动画
  • 导航
  • 存储
  • 网络
  • 于现有代码的整合
  • 没了

OK，这就是GUI，你得想到的想不到的应用程序背后就是这些东西。
